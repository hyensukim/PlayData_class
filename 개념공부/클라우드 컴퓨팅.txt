클라우드 컴퓨팅
- 초기 저장매체의 부족으로 인해 개발.
- 컴퓨팅 자원이 인터넷을 통해 공용으로 사용되는 개념.
- 개인용 컴퓨터의 대중화 및 인터넷의 발전으로 클라우드 컴퓨팅 기술이 성장.
- 가상화 기술(실제존재하지 않지만, 여러 운영체제나 소프트웨어를 시뮬레이션할 수 있는 기술)
- 하이퍼바이저 기술.(하나의 하드웨어 위에 하이퍼바이저를 통해 여러 OS 환경 사용)
- 컨테이너 가상화 기술.(하이퍼바이저의 기술 및 성능 한계로 나옴, 각각의 컴퓨팅 환경을 모듈화하여 사용하는 것)

클라우드 컴퓨팅 서비스
- 인프라 제공 _Iaas : 하드웨어
- 플렛폼 제공_Pass : 가상서버
- 소프트웨어 제공_Sass : 소프트웨어

클라우드 운영 방식
- 프라이빗 클라우드 : 특정 사용자만 사용.
- 퍼블릿 클라우드 : 불특정 다수가 자원을 공유하여 사용.
- 하이브리드 클라우드 : AWS

AWS 온라인 입문 교육
- AWS Builder
